# asn_cache/

Pre-computed per-ASN network range CSV files generated by `asn_cache_generator.py`.
Each file covers all network blocks for one ASN, deduplicated and enriched with country data.

Used by `asn_view.php` to serve ASN network listings offline without querying MMDB files at request time.

## File naming

```
{ASN}_{CC}_{Org}.csv
```

| Part  | Example               | Description                                      |
|-------|-----------------------|--------------------------------------------------|
| `ASN` | `AS12389`             | Autonomous System Number                         |
| `CC`  | `RU`                  | ISO 3166-1 alpha-2 country code of the majority of the ASN's networks |
| `Org` | `Rostelecom`          | Sanitised organisation name (max 40 chars, spaces â†’ underscores) |

Example: `AS12389_RU_Rostelecom.csv`

## CSV columns

```
network,ip_version,country,country_code,asn,org,source
1.2.3.0/24,IPv4,Russia,RU,AS12389,Rostelecom,ipinfo
```

| Column         | Description                                               |
|----------------|-----------------------------------------------------------|
| `network`      | CIDR block (e.g. `1.2.3.0/24`)                           |
| `ip_version`   | `IPv4` or `IPv6`                                         |
| `country`      | Full country name of the network block                    |
| `country_code` | ISO 3166-1 alpha-2 country code                          |
| `asn`          | ASN string (e.g. `AS12389`)                              |
| `org`          | Organisation / ISP name                                   |
| `source`       | MMDB source that won the conflict resolution (`ipinfo`, `iplocate`, or `maxmind`) |

## Prefix limits

Networks outside these ranges are excluded during generation:

| Family | Rule                             |
|--------|----------------------------------|
| IPv4   | `/1` â€“ `/24` (no host routes)    |
| IPv6   | `/32` â€“ `/48`                    |

## Generation

Run from the project root:

```bash
# Process all ASNs (skips existing files)
python3 asn_cache_generator.py

# Overwrite all existing files
python3 asn_cache_generator.py --force

# Limit parallel workers
python3 asn_cache_generator.py --workers 4

# Single ASN (useful for testing)
python3 asn_cache_generator.py --asn AS12389

# Use majority-vote conflict resolution instead of ipinfo-priority
python3 asn_cache_generator.py --vote
```

MMDB source files are read from the parent directory (`../`).
Output files are written here (`asn_cache/`).

## Conflict resolution

When the same CIDR block is claimed by different ASNs across the source databases, the winner is chosen by:

- **Default (ipinfo priority):** ipinfo > iplocate > maxmind
- **`--vote` mode:** majority wins; ties broken by the same priority order

## Source databases

| Source   | ASN file            | Country file            |
|----------|---------------------|-------------------------|
| ipinfo   | `ipinfo_lite.mmdb`  | `ipinfo_lite.mmdb`      |
| iplocate | `ip-to-asn.mmdb`    | `ip-to-country.mmdb`    |
| maxmind  | `GeoLite2-ASN.mmdb` | `GeoLite2-Country.mmdb` |

Missing database files are skipped automatically; at least one ASN source must be present.
